## Overview

AgentSpace uses specialized AI agents powered by the OpenAI Agents SDK to handle different types of tasks. Each agent is configured with specific instructions and expertise to provide accurate, focused responses in their domain.

Agents in AgentSpace are built using the `@openai/agents` package, which provides a simple yet powerful interface for creating autonomous AI assistants that can process requests and return intelligent responses.

## Agent Architecture

All agents in AgentSpace follow a consistent pattern:

```typescript
import { Agent, run } from '@openai/agents';

const myAgent = new Agent({
  name: 'AgentName',
  instructions: 'Specialized instructions for the agent',
});

const result = await run(myAgent, userInput);
```

The `run()` function executes the agent with the provided input and returns a result containing the `finalOutput` field with the agent's response.

    ### Implementation

    Located in `source/internal/agents/context-summarizer.ts:10-20`, this agent includes PDF processing capabilities:

    ```typescript
    export const getContextSummaryPostHandler = asyncHandler(async (req: Request, res: Response) => {
      const uploadedFile = req.file;

      if (!uploadedFile?.filename.endsWith('.pdf')) {
        res.status(400).json({ error: 'Only PDF files are supported' });
        return;
      }

      logger.info('Received file for context summarization:', uploadedFile?.originalname);

      try {
        const extractedData = await parsePDFBuffer(uploadedFile.path);

        if (!extractedData) {
          res.status(500).json({ error: 'Failed to extract text from PDF' });
          return;
        }

        const summaryPrompt = `Summarize the following research paper:\n\n${extractedData.text}`;
        const result = await run(contextAgent, summaryPrompt);

        res.status(200).json({ summary: result.finalOutput });
      } finally {
        await unlink(uploadedFile.path).catch(() => {});
      }
    });
    ```

    ### API Endpoint

    **POST** `/agents/context-summarizer`

    Content-Type: `multipart/form-data`

    Form field: `myFile` (PDF file, max 5MB)

    Response:
    ```json

    ```

    ### Features

    - Accepts PDF files up to 5MB
    - Extracts text using the `pdf-parse` library
    - Automatically cleans up uploaded files after processing
    - Provides structured summaries with research objectives, methods, findings, and implications

## Creating Custom Agents

To add a new agent to AgentSpace:

1. **Create the agent file** in `source/internal/agents/`:

```typescript
import { Agent, run } from '@openai/agents';
import type { Request, Response } from 'express';

const myCustomAgent = new Agent({
  name: 'MyCustomAgent',
  instructions: 'Your specialized instructions here',
});

export const getCustomHandler = async (req: Request, res: Response) => {
  const { input } = req.body;

  if (!input) {
    return res.status(400).json({ error: 'Input is required' });
  }

  try {
    const result = await run(myCustomAgent, input);
    return res.send({ result: result.finalOutput });
  } catch (error) {
    console.error(error);
    return res.status(500).json({ error: 'Internal server error' });
  }
};
```

2. **Register the route** in `source/internal/router/agents.ts`:

```typescript
import { getCustomHandler } from '../agents/my-custom-agent.js';

router.get('/custom', getCustomHandler);
```

3. **Test your agent** at `/agents/custom`

<Note>
  Agent instructions are crucial for performance. Be specific about the agent's expertise, expected input format, and desired output structure.
</Note>
