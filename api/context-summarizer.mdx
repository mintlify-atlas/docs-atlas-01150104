---
title: "Context summarizer"
description: "Analyze and summarize PDF research papers"
---

The context summarizer endpoint allows you to upload academic research papers in PDF format and receive comprehensive summaries. The agent extracts key information including research objectives, methodology, findings, and implications.

## Endpoint

```
POST /agents/context-summarizer
```

## Request Parameters

<ParamField body="myFile" type="file" required>
  The PDF research paper to analyze and summarize. 
  
  **Requirements:**
  - File format: PDF only (`.pdf` extension)
  - Maximum size: 5 MB
  - Field name must be `myFile`
</ParamField>

## Request Example

```bash
curl -X POST https://api.agentspace.com/agents/context-summarizer \
  -F "myFile=@/path/to/research-paper.pdf"
```

## Response Format

<ResponseField name="summary" type="string">
  A comprehensive summary of the research paper including:
  - The main research question or objective
  - Methods and approach used in the study
  - Key findings and results
  - Significance and implications of the research
  
  The summary is written in clear, plain language accessible to a general audience.
</ResponseField>

## Response Example

```json
{
  "summary": "This study investigates the impact of climate change on coral reef ecosystems in the Pacific Ocean. The researchers employed a combination of field observations, satellite imagery analysis, and predictive modeling over a 10-year period. Key findings reveal a 23% decline in coral coverage, with rising ocean temperatures being the primary driver. The study also identified several resilient coral species that showed adaptation to warmer conditions. These findings are significant for conservation efforts, suggesting that focused protection of resilient species and implementation of local cooling interventions could help preserve reef ecosystems. The implications extend to fisheries management and coastal protection strategies for island nations dependent on reef systems."
}
```

## File Requirements

<Warning>
  Only PDF files are accepted. Attempting to upload other file formats will result in a 400 error.
</Warning>

<Info>
  The uploaded file is automatically deleted from the server after processing, regardless of whether the summarization succeeds or fails.
</Info>

## Error Responses

<Accordion title="400 Bad Request - Missing File">
  Returned when no file is provided in the request.
  
  ```json
  {
    "error": "No file provided."
  }
  ```
</Accordion>

<Accordion title="400 Bad Request - Invalid Format">
  Returned when the uploaded file is not a PDF.
  
  ```json
  {
    "error": "Only PDF files are supported"
  }
  ```
</Accordion>

<Accordion title="400 Bad Request - File Too Large">
  Returned when the file exceeds the 5 MB size limit.
  
  ```json
  {
    "error": "File too large (max 5MB)."
  }
  ```
</Accordion>

<Accordion title="500 Internal Server Error - Extraction Failed">
  Returned when the PDF content cannot be extracted or parsed.
  
  ```json
  {
    "error": "Failed to extract text from PDF"
  }
  ```
</Accordion>

<Accordion title="500 Internal Server Error - Upload Failed">
  Returned when the file upload process encounters an unexpected error.
  
  ```json
  {
    "error": "File upload failed."
  }
  ```
</Accordion>

## Code Examples

<CodeGroup>

```javascript Node.js
const FormData = require('form-data');
const fs = require('fs');

const form = new FormData();
form.append('myFile', fs.createReadStream('/path/to/research-paper.pdf'));

const response = await fetch('https://api.agentspace.com/agents/context-summarizer', {
  method: 'POST',
  body: form
});

const data = await response.json();
console.log(data.summary);
```

```python Python
import requests

with open('/path/to/research-paper.pdf', 'rb') as file:
    files = {'myFile': file}
    response = requests.post(
        'https://api.agentspace.com/agents/context-summarizer',
        files=files
    )

data = response.json()
print(data['summary'])
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "mime/multipart"
    "net/http"
    "os"
)

func main() {
    file, _ := os.Open("/path/to/research-paper.pdf")
    defer file.Close()

    body := &bytes.Buffer{}
    writer := multipart.NewWriter(body)
    part, _ := writer.CreateFormFile("myFile", "research-paper.pdf")
    io.Copy(part, file)
    writer.Close()

    req, _ := http.NewRequest("POST", "https://api.agentspace.com/agents/context-summarizer", body)
    req.Header.Set("Content-Type", writer.FormDataContentType())

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]string
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result["summary"])
}
```
