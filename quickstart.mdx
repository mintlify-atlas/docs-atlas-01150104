<Steps>
  <Step title="Clone repository and install dependencies">
    First, clone the AgentSpace repository and install all required dependencies:

    ```bash
    git clone <repository-url>
    cd source
    npm install
    ```

    This will install all the necessary packages including Express.js, OpenAI Agents SDK, TypeScript, and development tools.
  </Step>

  <Step title="Set up environment variables">
    Create a `.env` file in the root directory and add your OpenAI API key:

    ```bash .env
    OPENAI_API_KEY=your_openai_api_key_here
    ```

    <Warning>
      You need a valid OpenAI API key to use AgentSpace. Get one from the [OpenAI Platform](https://platform.openai.com/api-keys).
    </Warning>
  </Step>

  <Step title="Run the development server">
    Start the development server with hot-reloading enabled:

    ```bash
    npm run dev
    ```

    The server will start on port `8090`. You should see:

    ```bash
    agentspace live at http://localhost:8090
    ```

    <Note>
      The dev command automatically formats code, builds TypeScript, and watches for changes.
    </Note>
  </Step>

  <Step title="Test your first agent">
    Test the translation agent with a simple curl command:

    ```bash
    curl -X GET http://localhost:8090/agents/translate \
      -H "Content-Type: application/json" \
      -d '{
        "question": "How do you say hello in French?"
      }'
    ```

    You should receive a response like:

    ```json
    {
      "answer": "In French, you say 'Bonjour' for hello."
    }
    ```
  </Step>
</Steps>

## Available Endpoints

Once your server is running, you can interact with these endpoints:

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/agents/translate` | GET | Translate text and answer language questions |
| `/agents/history` | GET | Answer questions about world history |
| `/agents/context-summarizer` | POST | Summarize PDF research papers |

## Testing the PDF Summarizer

To test the PDF summarizer agent, use a multipart/form-data request:

```bash
curl -X POST http://localhost:8090/agents/context-summarizer \
  -F "file=@/path/to/research-paper.pdf"
```

The agent will extract text from the PDF and return a structured summary with:
- Main research question or objective
- Methods and approach
- Key findings and results
- Significance and implications

## Request Format

For the translation and history agents, send JSON requests with a `question` field:

```json
{
  "question": "Your question here"
}
```

**Example: History Agent**
```bash
curl -X GET http://localhost:8090/agents/history \
  -H "Content-Type: application/json" \
  -d '{
    "question": "Who was the first emperor of Rome?"
  }'
```
